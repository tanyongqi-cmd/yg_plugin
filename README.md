# yg_plugin

## 安装
```
npm i yg_plugin
```
## 介绍

### ygDialog

|  方法名   | 说明  | 	参数  |
|  ----  | ----  | ----  |
| show  | 打开弹框 | (title,type,option,row) title=弹框标题 type=进入的组件名 option=弹框配置项 row=自定义参数

|  事件名称   | 说明  | 	回调参数  |
|  ----  | ----  | ----  |
| cancel  | 弹框关闭回调事件 | row 自定义回调参数

### ygRegionSelector

|  参数   | 说明  |
|  ----  | ----  |
| value / v-model	  | 地区id集合 |
| placeholder  | 提示文本 |
| options  | 未开发(暂不可用) |
